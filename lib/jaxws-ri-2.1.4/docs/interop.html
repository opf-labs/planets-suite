JAX-WS RI 2.1.4 provides a comprehensive interoperability platform to develop web services solutions. Here are the key interoperability features:

WSDL and Message Level Interoperability

  • JAX-WS 2.1
  • JAXB 2.1
  • SAAJ
  • Stax
  • WS-I Basic Profile 1.1
  • WS-I Simple SOAP Binding Profile 1.0
  • WS-I Attachment Profile 1.0
  • SOAP 1.1 and SOAP 1.2
  • WSDL 1.1
  • WS-Addressing  - Core, SOAPBinding and WSDL Binding
  • MTOM and SOAP With Attachments

.NET 3.0 Interoperability

  • Provided thru WSIT plugins.
  • JAX-WS 2.1.4 and WSIT plugins are tightly integrated in to GF v2
  • Tested for interoperability with .NET 3.0
  • .NET 3.0 interoperable WS-* features supported through WSIT
    • WS-ReliableMessaging
    • WS-Policy
    • WS-MEX
    • WS-Security/WS-Security Policy
    • WS-Trust/WS-Secure Conversation
    • WS-AT/WS-Tx (Supported only on Glassfish v2)
    • WS-Addresssing
    • MTOM
